Welcome to the Data Governance Monitor WebPages !

This project proposes to design a Data Governance monitor application. The goal of this application is not to cover all the data governance aspects but to have a simple and easy to use application which enables to collect information (data quality scores, business taxonomy, etc.) and give the ability to manipulate these in a kind of data governance datamart.
